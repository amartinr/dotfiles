# i3 configuration
## Requirements
* i3-gaps-git
* i3ipc-python-git
* i3lock-fancy-dualmonitors-git
* termite
* system-san-francisco-font-git
* ttf-dejavu-sans-mono-powerline-git
* ttf-ionicons
* ttf-font-awesome
* rofi
* feh
* gsimplecal
* pavucontrol
* connman
* scrot
* vim-colorschemes
* vim-airline
* vim-airline-themes
* vim-promptline-git
* pygtmentize
* pygments-base16
